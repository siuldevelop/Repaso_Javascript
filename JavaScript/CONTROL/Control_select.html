Colores de semaforo

El elemento select se utiliza para crear un menú desplegable en el que el usuario puede seleccionar una opción. Cada opción dentro del menú se define con el elemento option. El atributo value de cada opción se utiliza para almacenar un valor asociado a esa opción, que puede ser diferente del texto visible que se muestra al usuario.
